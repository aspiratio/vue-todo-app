# vue-todo-app

## Project Create

```sh
pnpm i @vue/cli
```

```sh
pnpm create vue vue-todo-app
```

## Project Setup

```sh
pnpm install
```

### Compile and Hot-Reload for Development

```sh
pnpm dev
```

### Type-Check, Compile and Minify for Production

```sh
pnpm build
```

### Lint with [ESLint](https://eslint.org/)

```sh
pnpm lint
```
